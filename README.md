**Inventra** is a desktop-based Product & Inventory Management application designed to simplify product tracking, inventory control, and sales operations for businesses. Built with Java and powered by SQLite, Inventra provides a modular, scalable, and user-friendly solution for efficient product management.

---

## Features

- **Product Management:** Add, update, and view products with detailed information.  
- **Inventory Tracking:** Monitor stock levels and receive alerts for low stock.  
- **Database Integration:** Persistent storage using SQLite for reliable and efficient data handling.  
- **Intuitive GUI:** Easy-to-use interface built with JavaFX for a seamless user experience.  
- **Design Patterns:** Implementation of Observer, Singleton, Memento, Command, Bridge, Builder, and Strategy design patterns to ensure modularity, scalability, and maintainability.  
- **Reporting:** Generate simple reports for inventory and product management.

---

## Technologies Used

- **Programming Language:** Java  
- **Build Tool:** Maven
- **Database:** SQLite   
- **GUI Framework:** JavaFX
